PHPLibrary
==========

for DataBase's homework and Recommend System

## 功能描述：

### 基本功能：

+ 1. 管理员登陆
+ 2. 图书入库
+ 3. 图书查询
+ 4. 借书
+ 5. 借书证管理

### 特别功能：


+ 1. 借书排序
	+ 被借排序
	+ 曾经借过书排序
	+ 
+ 2. 个人推荐
+ 3. 书籍评论
+ 4. 书籍标签
